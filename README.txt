-Pour lancer le programme vous devez faire un run a la classe RunProgram.java.
-Les vues principales de notre projet contient des methodes "main"
 pour faciliter leurs tests ,mais notez bien que il est possible d'avoir des erreurs
 si vous cliquez sur un button au niveau de la console par ce que les vues sont liees 
 entre eux avec des controlleurs, et ces methodes main sont juste pour afficher les vues.
-Dans le package DAO vous trouvez un class test.java pour tester les methodes de ce packages.
